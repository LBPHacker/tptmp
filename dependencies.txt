the server is meant to be run on lua 5.3 or above

lunajson
jnet
http
luafilesystem
basexx (comes with http)
cqueues (comes with http)
luaossl (comes with http)

thus:
luarocks install --lua-version=5.3 lunajson
luarocks install --lua-version=5.3 --server=https://luarocks.org/dev jnet
luarocks install --lua-version=5.3 http
luarocks install --lua-version=5.3 luafilesystem
